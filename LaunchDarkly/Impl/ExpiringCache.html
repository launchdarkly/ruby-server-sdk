Overview

This class is part of a private API. You should avoid using this class if possible, as it may be removed or be changed in the future.

A thread-safe cache with maximum number of entries and TTL. Adapted from https://github.com/SamSaffron/lru_redux/blob/master/lib/lru_redux/ttl/cache.rb under MIT license with the following changes:

  • made thread-safe
  • removed many unused methods
  • reading a key does not reset its expiration time, only writing

def initialize(max_size, ttl)
  @max_size = max_size
  @ttl = ttl
  @data_lru = {}
  @data_ttl = {}
  @lock = Mutex.new
end

def []=(key, val)
  @lock.synchronize do
    ttl_evict

    @data_lru.delete(key)
    @data_ttl.delete(key)

    @data_lru[key] = val
    @data_ttl[key] = Time.now.to_f

    if @data_lru.size > @max_size
      key, _ = @data_lru.first # hashes have a FIFO ordering in Ruby

      @data_ttl.delete(key)
      @data_lru.delete(key)
    end

    val
  end
end
